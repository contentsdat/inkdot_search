잉크닷 서치
사용법

공공기관 유튜브 아카이브 데이터를 검색하고 분석하는 세 가지 핵심 기능을 안내합니다.

01 키워드 검색 원하는 영상을 조건에 맞게 빠르게 찾기 02 비교 분석 두 기관의 유튜브 성과를 나란히 비교 03 분석 개별 기관 채널 심층 분석
01

키워드 검색

🔍

영상 키워드 검색

원하는 영상을 조건에 맞게 빠르게 찾기

01
검색어 입력

상단 검색창에 찾고 싶은 키워드를 입력합니다. 영상 제목이나 기관명 등 관련 단어를 자유롭게 입력하세요.

02
필터 설정

검색 범위를 좁히기 위해 아래 필터를 조합해 사용할 수 있습니다.

기관 구분
중앙 / 지방
영상 유형
일반 / 쇼츠
기관명
특정 기관 선택
수집 기간
기간 범위 지정
03
결과 정렬 및 확인

검색 결과는 아래 기준으로 정렬할 수 있습니다. 영상 카드를 클릭하면 유튜브로 바로 연결됩니다.

조회수 높은순 좋아요 높은순 댓글 많은순 최신순 오래된순
04
결과 내보내기

검색 결과 목록은 EXPORT 버튼을 통해 파일로 저장할 수 있습니다.

💡 팁 — 필터를 조합할수록 원하는 영상을 더 빠르게 찾을 수 있습니다. 예를 들어 "지방기관 + 쇼츠 + 특정 기관명"으로 좁혀보세요.
02

비교 분석

⇄

기관 간 비교 분석

두 기관의 유튜브 성과를 나란히 비교

01
비교할 기관 선택

왼쪽 드롭다운에서 기준 기관을, 오른쪽 드롭다운에서 비교할 기관을 각각 선택합니다. +VS 버튼으로 두 기관을 세팅하세요.

02
비교 기간 설정

시작 연도와 끝 연도를 지정해 원하는 기간 내 데이터만 비교할 수 있습니다.

03
비교 지표 선택

비교하고 싶은 지표를 복수 선택할 수 있습니다.

👁️
조회수
❤️
좋아요
💬
댓글
04
구독자 추이 함께 보기

지표 선택 버튼 옆의 구독자 버튼을 켜면 분석 결과 하단에 월별 구독자 추이 그래프가 추가됩니다. 그래프에 마우스를 올리면 해당 월의 구독자 수를 툴팁으로 확인할 수 있으며, 두 기관을 선택한 경우 색상으로 구분해 나란히 비교합니다.

05
영상 유형 필터 적용 후 분석 실행

일반 영상과 쇼츠를 구분하거나 전체를 함께 볼 수 있습니다. 설정 완료 후 분석 → 버튼을 클릭하면 결과가 표시됩니다.

💡 팁 — 유사한 규모나 분야의 기관끼리 비교하면 더 의미 있는 인사이트를 얻을 수 있습니다. 구독자 버튼은 분석 실행 전후 언제든지 토글할 수 있습니다.
03

구독자 분석

📈

월별 구독자 추이 분석

비교 탭에서 기관의 구독자 변화를 그래프로 확인

01
비교 탭에서 기관 선택

⇄ 비교 탭으로 이동한 뒤 구독자 추이를 확인하고 싶은 기관을 선택합니다. 단일 기관 또는 두 기관을 함께 선택해 비교할 수 있습니다.

02
구독자 버튼 활성화

지표 선택 영역 옆의 구독자 버튼을 클릭해 활성화합니다. 조회수·좋아요·댓글 지표 버튼과 독립적으로 동작하며, 분석 결과 하단에 구독자 그래프가 별도로 표시됩니다.

03
기간 설정 후 분석 실행

시작·끝 연도를 설정하면 해당 기간의 월별 구독자 추이만 그래프로 표시됩니다. 설정 완료 후 분석 → 버튼을 클릭하세요.

04
그래프 및 수치 확인

월별 구독자 수 추이가 꺾은선 그래프로 표시됩니다. 그래프 위에 마우스를 올리면 해당 월의 구체적인 구독자 수를 툴팁으로 확인할 수 있습니다. 두 기관을 선택한 경우 각 기관의 추이를 색상으로 구분해 나란히 비교합니다.

💡 팁 — 구독자 데이터는 매월 말 수집 기준입니다. 구독자 버튼은 분석 실행 전후 언제든지 토글할 수 있으며, 기간 필터도 영상 지표와 동일하게 적용됩니다.
03

분석

◈

채널 분석

개별 기관의 유튜브 채널 심층 분석

01
분석 페이지로 이동

상단 메뉴의 ◈ 분석 버튼을 클릭하면 inkdot.kr 분석 페이지로 이동합니다.

02
현재 결과 요약 확인

검색 또는 비교 설정 기준으로 집계된 전체 통계(STATS)가 상단에 표시됩니다. 총 영상 수, 합산 조회수, 좋아요, 댓글 등 핵심 수치를 한눈에 확인할 수 있습니다.

03
탑 채널 TOP 3 확인

조회수, 좋아요, 댓글 각 지표별로 상위 3개 채널을 확인할 수 있습니다. 어떤 기관이 어떤 지표에서 두각을 나타내는지 파악하세요.

04
개별 영상 상세 확인

목록에서 각 영상의 조회수·좋아요·댓글 수치를 확인할 수 있으며, 유튜브 바로가기 링크로 해당 영상을 직접 시청할 수 있습니다. 수치는 수집일 기준입니다.

💡 팁 — 분석 기능은 검색이나 비교에서 설정한 조건을 그대로 반영합니다. 먼저 검색/비교 조건을 세팅한 뒤 분석을 실행하면 더 정확한 결과를 확인할 수 있습니다.
